# 旅程管理應用

一個功能完整的旅程規劃和管理應用，包括行程規劃、景點介紹、住宿安排和**旅程記帳**功能。

## ✨ 主要功能

### 核心功能
- 📅 **行程規劃** - 按日期管理旅程行程
- 📍 **景點介紹** - 保存景點信息和推薦
- 🛏️ **住宿管理** - 記錄住宿地點和交通方式
- 💰 **旅程記帳** - 完整的記帳和清算系統

### 記帳功能 (NEW!)
- 🏷️ **快速記帳** - 8個分類快捷鍵
- 💵 **支付方式** - 現金、信用卡、IC卡
- 💱 **匯率管理** - 支持多幣別換算
- 👥 **分帳計算** - 自動計算分擔費用
- 🧮 **清算建議** - 自動生成清算報告
