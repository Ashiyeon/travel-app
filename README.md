# ✈️ 空姐叫我 - 旅程管理與共用協作平台

空姐叫我 是一個為旅人量身打造的現代化旅程規劃與管理應用。不僅能幫助您輕鬆安排每日行程、記錄必去景點與住宿資訊，更新增了強大的「**共同編輯**」與「**智慧分帳**」功能，讓您與旅伴能輕鬆共享行程，出遊算錢不再傷感情！

## ✨ 核心功能亮點

### 👥 共同編輯與權限管理 (NEW!)
- **一鍵邀請**：輸入旅伴的 Email，對方登入後即可在首頁看到該筆行程。
- **即時同步**：所有擁有權限的成員都能共同新增、編輯行程與記帳內容。
- **名單自動帶入**：記帳時，系統會自動抓取擁有該行程權限的真實帳號名單作為「付款人」與「分擔人」選項。

### 💰 智慧旅程記帳與分帳
- **快速記帳介面**：8 大分類快捷鍵 (餐飲、交通、住宿等) 手機操作更順手。
- **多幣別與即時匯率**：支援 JPY, USD, TWD 等常見幣別，自動換算台幣總額。
- **多元分帳模式**：支援「平分」、「自訂具體金額」、「依百分比」三種拆帳規則。
- **自動清算報告 (Settlement)**：系統自動計算「誰付了多少」、「誰該付多少」，並生成「誰欠誰錢」的最佳還款指示，讓旅伴間互不相欠。
- **視覺化支出圖表**：精美的甜甜圈圖 (Doughnut Chart) 讓您一眼看出花費最多的項目。

### 🗺️ 完善的行程規劃
- **📅 行程 (Itinerary)**：按日期與時間軸管理每日活動。
- **📍 景點 (Attractions)**：保存景點資訊、筆記與推薦度。
- **🛏️ 住宿 (Accommodation)**：記錄飯店名稱、入住/退房時間與交通方式。
